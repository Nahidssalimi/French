# French Repository

## Overview

This repository contains the HTML lesson files for the French curriculum. The content is organized by modules, units, and lessons to provide a structured learning experience.

If you need to edit a specific lesson, please locate the corresponding HTML file based on the naming convention described below. Files can be downloaded, edited locally, and then re-uploaded (hosted) to update the curriculum content.

---

## File Naming Convention

Each file name follows a standardized format:

**FMx-Uy-Lz.html**

Where:

* **F** = French
* **M** = Module
* **U** = Unit
* **L** = Lesson

### Examples

| File Name      | Meaning                           |
| -------------- | --------------------------------- |
| FM2-U1-L1.html | French Module 2, Unit 1, Lesson 1 |
| FM3-U4-L2.html | French Module 3, Unit 4, Lesson 2 |
| FM5-U2-L5.html | French Module 5, Unit 2, Lesson 5 |

---

## Editing Content

To update a lesson:

1. Locate the required file using the naming convention above.
2. Download the HTML file.
3. Make the necessary edits using your preferred HTML editor.
4. Save the updated file.
5. Upload the revised version to the repository.
6. Re-host or deploy the updated content if applicable.

---

## Repository Structure

The curriculum is organized into:

* **Modules** – Major learning sections covering broader themes and language objectives.
* **Units** – Groups of related lessons within a module.
* **Lessons** – Individual instructional pages delivered as HTML files.

---

## Best Practices

When editing files:

* Preserve the existing file naming structure.
* Maintain consistent formatting and styling.
* Verify that all links, images, and embedded resources continue to function correctly.
* Test the updated HTML file before deployment.
* Commit changes with clear and descriptive commit messages.

### Example Commit Messages

* `Updated vocabulary activity in FM3-U2-L4`
* `Fixed grammar explanation in FM4-U1-L2`
* `Added listening exercise to FM5-U3-L1`

---

## Support

When requesting changes, please reference the exact file name (for example: `FM4-U2-L3.html`) to ensure the correct lesson is updated.

This repository serves as the source for the French curriculum content and should be maintained with consistent structure and version control practices.
