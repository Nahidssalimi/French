Exercice 4 — Reliez et copiez la question complète

Lisez la fin de chaque question et choisissez la bonne lettre (a–g).

1. Vous êtes ouverts quels …
2. Quels sont …
3. Les promenades durent …
4. Est-ce que les chiens …
5. Quel est le …
6. Est-ce que c'est accessible …
7. Quelles sont les options …
Fins
a — les horaires d'ouverture?
b — combien de temps?
c — sont admis?
d — pour manger?
e — aux personnes handicapées?
f — prix d'entrée?
g — jours de la semaine?

Grammaire — Poser des questions avec quel

To ask which…? or what…?, use the adjective quel. It must agree with the noun.

masc sgfém sgmasc plfém pl
quelquellequelsquelles
  • Quel est le prix d'une visite guidée?
  • Quelle heure est-il?
  • Quels sont les moyens de transport ici?
  • Quelles sont les activités culturelles que vous proposez?

À toi! — Complète avec quel / quelle / quels / quelles

1. est le tarif réduit?
2. est l'adresse de l'office de tourisme?
3. sont les jours d'ouverture?
4. sont les activités pour enfants?
5. À heure ferme l'aquarium?
6. bus va au château?
